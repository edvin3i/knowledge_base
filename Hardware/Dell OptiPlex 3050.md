# Dell OptiPlex 3050

Компактный десктоп, использую как ноды [[K3s]] кластера.

## Характеристики

| Параметр | Значение |
|----------|----------|
| **CPU** | Intel Core i5-7500T (4C/4T, 2.7-3.3 GHz) |
| **RAM** | 8 GB DDR4 |
| **Storage** | SSD 256 GB |
| **Form Factor** | Micro (USFF) |
| **TDP** | 35W |

## Использование

3 штуки как control-plane ноды [[K3s]]:

| Имя | IP |
|-----|-----|
| [[polynode-1]] | 192.168.20.221 |
| [[polynode-2]] | 192.168.20.222 |
| [[polynode-3]] | 192.168.20.223 |

## Почему OptiPlex

- Низкое энергопотребление (35W TDP)
- Компактный размер
- Тихий
- Надёжный (бизнес-линейка)
- Дешёвый на вторичке
