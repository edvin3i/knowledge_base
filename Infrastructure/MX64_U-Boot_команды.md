# MX64 — U-Boot команды

Справочник команд U-Boot для [[Cisco Meraki MX64]].

## USB

| Команда | Описание |
|---------|----------|
| `usb start` | Инициализация USB |
| `fatls usb 0:1` | Список файлов на флешке |
| `fatload usb 0:1 0x90000000 file.bin` | Загрузка файла в память |

## Загрузка

| Команда | Описание |
|---------|----------|
| `bootbk 0x90000000 kernel` | Запуск Meraki bootkernel |
| `bootm` | НЕ работает с форматом MX64 |
| `bootz` | НЕ работает с форматом MX64 |

## Диагностика

| Команда | Описание |
|---------|----------|
| `md 0x18000000 1` | Проверка версии SoC |
| `md 0x90000000 10` | Просмотр заголовка образа |
| `printenv` | Переменные окружения |
| `help` | Список команд |

## Проверка образа

```
u-boot> md 0x90000000 10
```

Правильный заголовок (Meraki bootkernel):
```
90000000: 4d495053 60008000 00500000 60008000    SPIM....P....
```

`4d495053` = "MIPS" (little-endian: "SPIM")
