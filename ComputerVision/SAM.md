# SAM

Segment Anything Model — foundation model для сегментации от Meta.

## Теория: Что такое SAM

### Назначение

**SAM (Segment Anything Model)** — универсальная модель сегментации:
- Zero-shot сегментация любых объектов
- Работа с подсказками: точки, bbox, текст
- Сегментация и трекинг в видео
- Авто-разметка датасетов

### Эволюция

| Версия | Год | Ключевое |
|--------|-----|----------|
| SAM | 2023 | Первая foundation model для сегментации |
| SAM 2 | 2024 | Видео-трекинг, улучшенная точность |
| SAM 3 | 2025 | Текстовые запросы, авто-разметка, тонкая настройка |

## SAM 3

### Ключевые возможности

- **Promptable сегментация**: точки, bbox, текстовые запросы
- **Видео-трекинг**: сегментация и отслеживание объектов в видео
- **Автоматическая разметка**: генерация масок для целых датасетов
- **Fine-tuning**: тонкая настройка на специфических доменах
- **API + локальный деплой**: через Roboflow Inference

### Применение в пайплайне

```
Данные → SAM 3 (авто-разметка) → CVAT (коррекция) → YOLO/RF-DETR (обучение) → DeepStream (деплой)
```

SAM 3 может значительно ускорить процесс разметки, создавая первичные маски, которые затем корректируются в [[CVAT]].

## Связано с
- [[YOLO]] — модели для обучения после разметки
- [[DETR]] — альтернативные детекторы/сегментаторы
- [[CVAT]] — ручная коррекция разметки
