# Признание диплома во Франции

Процедура признания российского диплома специалиста.

## Ключевые факты

| Параметр | Значение |
|----------|----------|
| **Орган** | [[ENIC-NARIC]] France |
| **Документ** | Attestation de comparabilité |
| **Стоимость** | €120 (€20 + €100) |
| **Срок** | 4-6 месяцев |
| **Обязательность** | Зависит от цели |

## Диплом специалиста → какой уровень?

| Французская система | Европейская рамка |
|---------------------|-------------------|
| **Bac+5** | **Niveau 7 (EQF)** |

> ⚠️ Итоговый уровень определяет эксперт ENIC-NARIC индивидуально

## Когда нужен ENIC-NARIC

| Цель | ENIC-NARIC |
|------|------------|
| [[Passeport Talent]] | Рекомендуется, не обязателен |
| Натурализация | **Не принимается с 2026!** |
| Поступление в вуз | Не нужен (вуз оценивает сам) |
| Трудоустройство | На усмотрение работодателя |

## Процедура

### 1. Подготовка в России

**Апостиль на диплом:**
- Орган: **Рособрнадзор**
- Срок: 1-2 месяца
- Бессрочный
- Только в России!

**Перевод:**
- [[Присяжный переводчик]] (traducteur assermenté)
- Диплом + приложение с оценками

### 2. Подача онлайн

**Сайт:** [phoenix.france-education-international.fr](https://phoenix.france-education-international.fr/inscriptions)

1. Регистрация → номер досье
2. Загрузка документов
3. Оплата €20 → проверка приемлемости
4. Оплата €100 → экспертиза
5. Получение attestation (PDF)

### 3. Необходимые документы

- Паспорт (скан)
- Диплом + апостиль
- Приложение к диплому
- Перевод диплома
- Перевод приложения

## Особые случаи

### Регулируемые профессии

ENIC-NARIC **не признаёт**:
- Медицина, фармацевтика
- Ветеринария
- Архитектура
- Юриспруденция

→ Отдельные процедуры через министерства

### Беженцы

- Бесплатно
- Приоритетное рассмотрение
- Можно без апостиля
